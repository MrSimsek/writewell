## Getting Started

### Prerequisites

- Node.js 18+ and npm

### Development

```bash
npm run dev
```
Open [http://localhost:3000](http://localhost:3000) in your browser.

### Building

```bash
npm run build
npm run start
```

## Project Structure

- `app/` - Next.js app directory
- `components/` - React components
- `contexts/` - React context providers
- `lib/` - Utility functions
- `types/` - TypeScript type definitions
